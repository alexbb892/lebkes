3. PANDUAN PENGEMBANGAN (Berdasarkan Rules)
------------------------------------------------------
- Standard File  : Selalu tambahkan baris `defined('BASEPATH') OR exit('No direct script access allowed');` di atas file Controller/Model baru.
- Keamanan DB    : Wajib menggunakan Query Builder CI3 (contoh: `$this->db->insert()`, `$this->db->where()`) untuk menghindari SQL Injection.
- Input & XSS    : Gunakan fitur `form_validation` dan filter `xss_clean` saat menerima input dari user. Pada view gunakan `html_escape()` saat menampilkan data rawan.
- Frontend UI    : Desain wajib responsif di semua devais (mobile, tablet, desktop) menggunakan sistem Grid Bootstrap 5. Jangan gunakan struktur tabel untuk layout tampilan, gunakan div flex/grid.
- Interaksi      : Gunakan jQuery AJAX untuk memproses data dari sisi client tanpa harus me-reload halaman sepenuhnya. Logika Controller menangkap request via `$this->input->is_ajax_request()`.
